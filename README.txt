Shanghai Travel Guide - Final Project Website
============================================

How to use:
1. Open index.html in Chrome to preview the website.
2. Replace placeholder images in the images folder with real Shanghai photos if you want.
3. If your real photos are JPG files, update the image paths in the HTML/CSS from .jpg to .jpg.
4. Upload the entire folder to GitHub Pages, InfinityFree, Netlify, or another free host.
5. Submit a ZIP file of this folder and paste the direct link to index.html in the comments box.

Files included:
- index.html
- attractions.html
- food.html
- itinerary.html
- contact.html
- css/style.css
- js/script.js
- images/*.jpg placeholder images

Assignment requirements covered:
- At least five HTML pages
- Shared CSS and JavaScript files
- Paragraphs, divs, spans, lists, images, table
- Fixed footer on every page
- Navigation bar connecting pages
- Bootstrap contact form
- Contact form message on submit
- Extra JavaScript interactivity: random travel tip, night mode, budget note
- CSS class selector, ID selector, pseudoclasses
- CSS Grid and Flexbox layout
- CSS transforms and filters
- CSS positioning
- Responsive design with viewport and media queries

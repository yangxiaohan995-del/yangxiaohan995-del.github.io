Image files used by this website
=================================

The website includes SVG placeholder images so it can run immediately.
For a better final project, replace these placeholders with real Shanghai photos.

Current image filenames used in the code:
1. shanghai-skyline.svg      - Shanghai skyline / Pudong skyline / night skyline
2. the-bund.svg              - The Bund riverfront
3. yu-garden.svg             - Yu Garden or City God Temple area
4. lujiazui.svg              - Lujiazui skyscrapers / Oriental Pearl Tower
5. shanghai-disney.svg       - Shanghai Disney Resort or theme park entrance
6. french-concession.svg     - Former French Concession / Wukang Road / tree-lined street
7. shanghainese-food.svg     - A table of Shanghainese dishes
8. xiaolongbao.svg           - Xiaolongbao soup dumplings
9. shengjianbao.svg          - Shengjianbao pan-fried buns
10. scallion-noodles.svg     - Scallion oil noodles
11. shanghai-metro.svg       - Shanghai metro station / metro train

Recommended real-photo size: at least 1200 x 800 pixels.
If your real photos are JPG files, put them into the images folder and update the .svg extensions in HTML/CSS to .jpg.
